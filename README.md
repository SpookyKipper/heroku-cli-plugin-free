# Usage
<!-- usage -->
```sh-session
$ npm install -g @heroku-cl/plugin-free
$ oclif-example COMMAND
running command...
$ oclif-example (-v|--version|version)
heroku-cli-plugin-free/0.0.0 linux-x64 node-v14.17.3
$ oclif-example --help [COMMAND]
USAGE
  $ oclif-example COMMAND
...
```
